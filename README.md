
# ALX System Engineering & DevOps

This repository contains projects completed as part of the ALX System Engineering & DevOps program.

## Directories

-   **0x00-shell_basics:** Introduction to the basics of shell scripting and commands.
-   **0x01-shell_permissions:** Introduction to file permissions and user management in shell.
-   **0x02-shell_redirections:** Introduction to shell redirections and filters.
-   **0x03-shell_variables_expansions:** Introduction to shell variables and expansions.

Each directory contains a series of scripts with exercises designed to introduce the basics of system engineering and DevOps.

## Usage

To use the scripts in this repository, clone the repository to your local machine:

bashCopy code

    `git clone https://github.com/BKaoukine/alx-system_engineering-devops.git` 

Then, navigate to the directory of the desired script and run it using the command `./<script-name>`.
